Scroll to Top

A JS/jQuery scroll-to-top script (adjusted from the one used in the WP Astra theme) that doesn't intefere with the functioning of :target and which isn't impaired by using CSS scroll-behavior.
Consists of the following parts:

  • CSS in the header to light up the target and position the scroll-to-top button
  • A little HTML markup for the scroll-to-top button
  • Load jQuery (CDN) to make the script work
  • Script in the footer with the scrolling and show and hide functionality.

NOTES: You can look at the source code or go to Github.com/Blindeman, where I store things I use regularly for safekeeping. CSS scroll-behavior doesn't work in Edge, IE or Safari but the anchor link and :target do work so I think that's acceptable. The scroll-to-top script scrolls the page and shows and hides the button depending on whether you've scrolled down or not. I've been thinking of just using CSS smooth-scroll for the scroll-to-top part too but for now this is what it is. I'm loading jQuery in the footer, but that means you get a flash of the button before it disappears again. Up to you.
